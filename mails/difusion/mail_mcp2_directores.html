Project MCP (Sesión 2): Supabase desde 0

¿Quieres aprender a construir un backend real en Supabase y entender por qué se ha convertido en una de las plataformas más usadas para productos modernos?
Este taller es la segunda sesión de una serie de 3 talleres donde iremos de 0 a 1: en la sesión 1 entendimos el Model Context Protocol (MCP), en esta sesión construiremos un proyecto completo en Supabase para un caso real de negocio, y en la sesión 3 crearemos un MCP sobre ese proyecto para conectar cualquier interfaz de IA con nuestro backend.
En esta sesión aprenderás:

  • Qué es Supabase y cómo funciona: Postgres como núcleo, arquitectura general y cómo se integra en productos reales.
  • Herramientas clave del ecosistema: Auth, Database, Storage, SQL Editor, API auto-generada, Edge Functions y gestión desde el Dashboard.
  • Seguridad y permisos: introducción práctica a RLS (Row Level Security), políticas y buenas prácticas para no romper seguridad.
  • Construcción del proyecto (caso real de negocio): diseñaremos el modelo de datos (tablas + relaciones), definiremos flujos básicos y dejaremos un backend listo para usarse en un producto.
  • Preparación para la sesión 3: dejaremos el proyecto "MCP-ready" para que una IA pueda consultarlo y ejecutar acciones de forma controlada.

Recomendable traer portátil. No hace falta ser experto: el objetivo es entenderlo bien y salir con un proyecto funcional.
